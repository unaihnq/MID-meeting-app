# 🚀 MID Meeting App

**최적의 미팅 장소를 찾는 지도 기반 PWA (Progressive Web App)**

## 📋 개요

MID Meeting App은 카카오맵 API를 활용하여 실시간으로 주변 카페, 레스토랑 등 미팅 장소를 검색하고 찾을 수 있는 모던한 웹 애플리케이션입니다.

## ✨ 주요 기능

### 🗺️ 실시간 지도
- 카카오맵 통합
- 마커 표시 및 인포윈도우
- 지도 레벨 조절

### 🔍 장소 검색
- 키워드 기반 검색
- 현재 위치 기반 검색 (GPS)
- 거리순 정렬
- 전화번호 및 주소 표시

### 📱 PWA 기능
- **홈 화면 설치** - 앱 처럼 사용 가능
- **오프라인 지원** - 서비스 워커를 통한 캐싱
- **빠른 로딩** - 최적화된 캐시 전략
- **다크모드** - 시스템 설정 자동 적용

### 📊 사용자 경험
- 반응형 디자인 (모바일/태블릿/PC)
- 직관적인 UI/UX
- 로딩 인디케이터
- 실시간 알림

## 📦 설치 및 실행

### 1️⃣ 로컬 테스트

Python이 설치되어 있다면:

```bash
# 프로젝트 디렉토리로 이동
cd mid-meeting-app

# 로컬 서버 실행
python3 -m http.server 8000

# 브라우저에서 열기
# http://localhost:8000
```

또는 Node.js http-server 사용:

```bash
npx http-server
```

### 2️⃣ GitHub Pages 배포

1. **저장소 Settings로 이동**
2. **Pages 섹션 선택**
3. **Source: main / / (root) 선택**
4. **Save 클릭**

자동으로 배포됩니다! 📤

배포 URL: `https://unaihnq.github.io/MID-meeting-app/`

### 3️⃣ 앱 설치 (모바일)

1. **배포된 URL 방문**
2. **브라우저 메뉴에서 "설치" 또는 "앱 설치" 선택**
3. **또는 "설치" 버튼 클릭 (iOS는 공유 → 홈 화면에 추가)**
4. **홈 화면에서 앱 실행**

## 🔑 API 설정

### Kakao Maps API 키

현재 앱에는 기본 API 키가 포함되어 있습니다. 커스텀 키를 사용하려면:

1. [Kakao Developers](https://developers.kakao.com) 방문
2. 계정 생성 및 로그인
3. 애플리케이션 등록
4. REST API 키 발급
5. `MID_meeting_app.html`의 다음 라인 수정:

```html
<script src="//dapi.kakao.com/v2/maps/sdk.js?appkey=YOUR_API_KEY&libraries=services"></script>
```

## 🎨 기능 사용법

### 장소 검색

1. 검색 입력창에 검색어 입력 (예: "카페", "레스토랑")
2. "🔍 검색" 버튼 클릭 또는 Enter 키 입력
3. 지도와 목록에 결과 표시

### 현재 위치 기반 검색

1. "📌 현재위치" 버튼 클릭
2. 위치 권한 허용
3. 자동으로 현재 위치 중심 카페 검색 실행

### 장소 상세보기

1. 목록에서 장소 카드 클릭
2. "🔗 상세보기" 링크 클릭하여 카카오맵 페이지 이동

## 🔄 오프라인 모드

Service Worker를 통해 오프라인 상황 지원:

- **첫 방문**: 필요한 파일 캐싱
- **오프라인**: 캐시된 파일 사용
- **외부 API**: 네트워크 필요 (Kakao Maps)

### 앱이 설치되지 않음

- ✅ HTTPS 사용 확인 (로컬: localhost 제외)

- ✅ GPS 활성화 확인
